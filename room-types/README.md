# 房型文章管理架構說明

## 📁 資料夾結構

```
room-types/
├── apartment.html          # 公寓式房型主頁
├── hotel-type.html         # 飯店式房型主頁
├── villa.html              # Villa式房型主頁
├── articles-list.html      # 文章列表頁面（範例：公寓式）
└── articles/               # 文章詳細頁面資料夾
    ├── article-template.html  # 文章模板
    ├── article-001.html       # 實際文章範例
    ├── article-002.html
    └── ...
```

## 🎯 建議的架構方案

### 方案一：每個房型一個文章列表頁面（推薦）

**優點：**
- 結構清晰，易於管理
- 每個房型的文章獨立分類
- 適合大量文章

**結構：**
```
room-types/
├── apartment.html
├── apartment-articles.html    # 公寓式文章列表
├── hotel-type.html
├── hotel-articles.html        # 飯店式文章列表
├── villa.html
├── villa-articles.html        # Villa式文章列表
└── articles/
    ├── apartment/
    │   ├── article-001.html
    │   ├── article-002.html
    │   └── ...
    ├── hotel/
    │   ├── article-001.html
    │   └── ...
    └── villa/
        ├── article-001.html
        └── ...
```

### 方案二：統一文章列表 + 分類篩選

**優點：**
- 所有文章集中管理
- 可以跨房型搜尋
- 適合文章數量較少時

**結構：**
```
room-types/
├── all-articles.html      # 統一文章列表
└── articles/
    ├── article-001.html   # 用標籤區分房型
    ├── article-002.html
    └── ...
```

## 📄 文章命名規範

### 建議格式
```
article-{房型代碼}-{序號}-{簡短標題}.html

範例：
- article-apt-001-choosing-guide.html
- article-hotel-001-review.html
- article-villa-001-tips.html
```

### 房型代碼
- `apt` = 公寓式 (Apartment)
- `hotel` = 飯店式 (Hotel)
- `villa` = Villa式

## 🎨 文章模板使用

### 基本結構
1. **文章標題和元資訊**
   - 標題
   - 分類標籤
   - 發布日期
   - 閱讀時間

2. **文章內容**
   - 特色圖片
   - 正文內容
   - 章節標題
   - 列表和引用

3. **文章結尾**
   - 標籤
   - 分享按鈕
   - 相關文章
   - 導航按鈕

### 複製模板步驟
1. 複製 `articles/article-template.html`
2. 重新命名為新文章
3. 修改標題、內容、標籤
4. 更新相關連結

## 🔗 連結關係

```
hotel.html
  └─> apartment.html (房型主頁)
        └─> articles-list.html (文章列表)
              └─> articles/article-001.html (文章詳細)
```

## 💡 最佳實踐建議

### 1. **內容規劃**
- 每篇文章專注一個主題
- 保持文章長度適中（1000-2000字）
- 使用清晰的章節結構

### 2. **SEO 優化**
- 每篇文章設定獨特的 meta 描述
- 使用相關關鍵字
- 添加結構化資料（可選）

### 3. **圖片管理**
- 為每篇文章準備特色圖片
- 圖片尺寸建議：1200x630px
- 使用描述性的 alt 文字

### 4. **更新維護**
- 定期更新過時資訊
- 檢查連結是否有效
- 更新價格和日期資訊

### 5. **用戶體驗**
- 提供清晰的導航路徑
- 相關文章推薦
- 易於分享的按鈕

## 🚀 快速開始

1. **創建新文章**
   ```bash
   複製 articles/article-template.html
   重新命名為 article-XXX.html
   編輯內容
   ```

2. **添加到文章列表**
   - 在 `articles-list.html` 中添加新的文章卡片
   - 設定正確的分類和標籤
   - 更新文章計數

3. **更新房型主頁**
   - 在房型主頁添加最新文章推薦
   - 更新文章列表連結
